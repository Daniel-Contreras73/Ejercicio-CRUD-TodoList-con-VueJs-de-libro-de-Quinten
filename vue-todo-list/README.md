# Vue 3 Todo List

Ejercicio de la página 13 del libro **"Vue 3 by Example"** de Joran Quinten.

## Descripción

Aplicación de lista de tareas (Todo List) construida con Vue 3 y Vite usando la Composition API.

## Funcionalidades

- ✅ Agregar tareas con Enter o botón "Add"
- ✅ Marcar tareas como completadas
- ✅ Eliminar tareas individuales
- ✅ Filtrar por: Todas / Activas / Completadas
- ✅ Limpiar tareas completadas
- ✅ Contador de tareas pendientes

## Tecnologías

- Vue 3 (Composition API)
- Vite
- JavaScript (ES6+)

## Instalación y uso

```bash
npm install
npm run dev
```

Abre [http://localhost:5173](http://localhost:5173) en tu navegador.
